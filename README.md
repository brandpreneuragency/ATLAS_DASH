# ATLAS DASH

ATLAS DASH is a local-first productivity and AI workspace that combines document editing, task management, CRM, lead forms, and live Hermes agent access in one interface.

The same React frontend can run as:

- a **Tauri desktop application** with native filesystem access, secure credential storage, file associations, and updater support;
- a **browser dashboard** backed by a loopback-only Node API, Caddy, Docker
